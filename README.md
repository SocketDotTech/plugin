### Installation

The widget can be installed via NPM or Yarn.

**NPM :**

```bash
npm install @socket.tech/widget
```

**Yarn :**

```bash
yarn add @socket.tech/widget
```

**Note :** The widget requires `react` (>=17.0.1) and `react-dom` (>=17.0.1) as peerDepencies.
